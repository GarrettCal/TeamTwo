User stories:
#1:  
As a user, I would like to create tasks =, edit them and delete them
#2:
As a user, I'd like to be able to login to an account with my tasks, so I can access them between devices.
#3:
As a user, I'd like to be able to mark a task as complete so it doesn't get mixed in with what I have in progress.
#4:
As a user, I'd like to be able to filter tasks, so I can see just the tasks I'm currently working on, or review tasks I've already completed.
#5:
As a user, I'd like to be able to upload task lists from external sources.
#6:
As a user, I'd like to be able to color code the tasks so that they stick out.
#7:
As a user, I'd like to be able to add subtasks to each task to allow myself to break down large tasks.
#8:
As a user, I'd like to be able to share my task list with other people.
#9:
As a user, I'd like to be able to create tasks that have a title, description, and due date, so I can keep track of what needs done.
#10:
As a user, I'd like to be able to edit tasks, so when plans change I don't have to create an entirely new task to replace it.
#11:
As a user, I'd like to be able to delete tasks, so I don't have to see tasks that aren't relevant anymore.
